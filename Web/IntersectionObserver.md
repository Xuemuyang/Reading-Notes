# IntersectionObserver

intersecton n. 交叉

这个 API 自动观察元素与某个区域间的交叉，在交叉不同程度(比率)时候触发 callback。

![](http://www.ruanyifeng.com/blogimg/asset/2016/bg2016110202.png)

## 应用

- 懒加载
- 曝光打点
