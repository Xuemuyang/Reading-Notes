## 记录

二进制位运算符用于直接对二进制位进行计算，一共有 7 个。

1. 按位与(AND) `&`
2. 按位或(OR) `|`
3. 按位异或(XOR) `^`
4. 按位非(NOT) `~`
5. 左移(Left shift) `<<`
6. 右移(Right shift) `>>`
7. 无符号右移 `>>>`

### 实践

#### `&` 判断奇偶数

```js
// 偶数 & 1 = 0
// 奇数 & 1 = 1
console.log(0 & 1)    // 0
console.log(1 & 1)    // 1
console.log(2 & 1)    // 0
console.log(3 & 1)    // 1
```

#### 取整

两种方式都是将小数直接舍去。

> 使用二进制否运算取整，是所有取整方法中最快的一种。 - 网道 - 二进制位运算符

```js
2.9 | 0 // 2
-2.9 | 0 // -2

~~2.9 // 2
~~47.11 // 47
~~1.9999 // 1
~~3 // 3

console.log(11.71 >> 0)  // 11
console.log(11.71 << 0)  // 11
console.log(11.71 >>> 0) // 11
```

#### 计算 2 的 n 次方

```js
1 << 2  // 4, 即 2的2次方
1 << 10 // 1024, 即 2的10次方
```

#### 判断符号是否相同

```js
(a ^ b) >= 0; //  true 相同; false 不相同
```
